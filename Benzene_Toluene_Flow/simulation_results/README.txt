This folder contains the results of the 2 by 2 design simulation.
The results are saved in 'results.pkl' file.
Each entry in the pickle file is a dictionary with the following keys:
 - 'Temp Heater': The temperature set for the heater.
 - 'Pressure Flash': The pressure set for the flash unit.
 - 'HeaterInputs': The inputs of the heater block.
 - 'VaporOutputs': The outputs of the vapor stream.
 - 'LiquidOutputs': The outputs of the liquid stream.
